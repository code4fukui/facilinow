# 福井県施設ダッシュボード

福井県の施設の開館状況や混雑状況を表示するウェブアプリです。

## デモ
https://code4fukui.github.io/facilinow/

## 機能
- 施設の開館状況や混雑状況の表示
- 施設ごとの詳細情報の表示
- ゲンキーの店舗情報のマップ表示

## 必要環境
特に必要ありません。ウェブブラウザ上で動作します。

## 使い方
ウェブブラウザ上で https://code4fukui.github.io/facilinow/ を開くだけで利用できます。

## データ・API
- 福井県の施設情報: 福井県庁DX推進課のオープンデータ
- 国民の祝日情報: 内閣府のオープンデータ
