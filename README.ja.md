# facilinow

福井県の公共施設のリアルタイムな状況を表示するダッシュボードと、ドラッグストアチェーン「ゲンキー」の店舗検索マップです。

> [!WARNING]
> 福井県施設ダッシュボード（メインアプリケーション）は、2024年3月31日をもって運用を終了しました。

## デモ

- **福井県施設ダッシュボード**: https://code4fukui.github.io/facilinow/
- **ゲンキーマップ**: https://code4fukui.github.io/facilinow/genkymap.html

## 機能

### 福井県施設ダッシュボード (`index.html`)

- 福井県の施設の開館状況（開館/閉館）、混雑状況、施設名、開館時間を表形式で表示します。
- 定休日、臨時休館日、および日本の国民の祝日に基づいて、現在の開館/閉館状況を自動的に計算します。
- 各施設の公式ウェブサイトおよびソースとなるオープンデータCSVへのリンクを提供します。

### ゲンキーマップ (`genkymap.html`)

- ゲンキーの店舗位置を示すインタラクティブなマップです。
- チェックボックスを使用して、都道府県ごとに店舗を絞り込むことができます。
- 市区町村別の店舗密度を色分けして視覚化したカラム地図（Tabular Map）を表示します。
- クリック可能なマップマーカーから、詳細な店舗情報ページへリンクします。
