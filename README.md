# facilinow
Fukui Prefecture Facility Dashboard

## Features
- Displays the opening status, congestion status, facility name, and opening hours of facilities in Fukui Prefecture.
- Provides a link to the open data CSV file for the facility information.
- Includes links to the data sources and developer information.

## Data / API
- Facility data: CC BY [Fukui Prefectural Government DX Promotion Division](https://www.pref.fukui.lg.jp/doc/dx-suishin/facilinow.html)
- Holiday data: CC BY Cabinet Office [About National Holidays - Cabinet Office](https://www8.cao.go.jp/chosei/shukujitsu/gaiyou.html)
